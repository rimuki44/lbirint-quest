# Лабиринт-квест

Текстовый квест на C++. Нужно решить головоломки и выбраться из лабиринта.

## Команды
- `osmotret` — посмотреть текущую локацию
- `idti N` — перейти в локацию N
- `otkrit N` — открыть локацию N (решить головоломку)
- `pomoc` — показать помощь
- `vihod` — выйти из игры

## Локации
1. **СТАРТ** — открыта
2. **МАТЕМАТИКА** — пример на сложение
3. **УГАДАЙ** — угадать число от 1 до 5
4. **ПОБЕДА** — финал

## Сборка и запуск
```bash
g++ main.cpp -o game
./game
